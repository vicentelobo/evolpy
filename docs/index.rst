Welcome to EvolPy's documentation!
==================================

EvolPy is an open-source Python library to use evolutionary algorithms for optimization problems. It seeks to be a simple, but powerful library allowing rapid testing. 

The following documentation presents all the options you can use to build your own evolutions.

The library is under continuous development. Feedback, bug reports, and contributions are very welcome! Our code is on `GitHub`_.
